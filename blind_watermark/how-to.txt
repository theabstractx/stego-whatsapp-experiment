It is assumed that all the files are in the same folder.

1. install blind_watermark using pip install blind-watermark

2. Embed the message
python embed.py --input cover.jpg --output stego.jpg --file secret.txt --wm_len_out wm_len.txt

3. Send stego.jpg via Whatsapp and download it again as received.jpg

4. Extract the message
python extract.py --input received.jpg --wm_len_in wm_len.txt --output extracted.txt

5. Evaluate
python evaluate.py stego.png received.jpg secret.txt extracted.txt
