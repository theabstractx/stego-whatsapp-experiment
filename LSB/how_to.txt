It is assumed that all the files are in the same folder.

1. Install python dependencies:
pip install pillow bitarray scikit-image

2. Embed the message
python lsb_embed.py cover.png secret.txt stego.png

(cover.png is the original picture)

3. Send stego.png via Whatsapp and download it again as received.jpg

4. Extract the message
python lsb_extract.py received.jpg extracted.txt

5. Evaluate
python evaluate.py cover.png received.jpg secret.txt extracted.txt

jpg conversion to access the LSB values happens automatically with pillow.
